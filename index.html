Le projet

Ce projet porte sur la visualisation des données économiques de l'espace CEDEAO, de 2005 à 2019.
Les éléments contenus dans les onglets des diagrammes vous mèneront à la visualisation.

Pour les diagrammes radar, le pays de référence choisi est le Nigéria.

DIAGRAMMES

Comme la Science des Données le veut, nous utilisons des diagrammes pour une meilleure visualisation des données.

OUTILS WEB

Pour sa réalisation, nous avons utilisé des outils de développement web, comme HTML, JavaScript, CSS et D3.js

APPRENTISSAGE

Ce projet nous a été donné dans le but de nous initier à la représentation des données dans le cadre de notre Master en Science des Données et Applications

VUE GLOBALE

Représentation de toutes les données sans catégorisation.

CATEGORISATION

Représentation des données avec une uniformisation, une expression de chaque valeur par rapport au maximum de sa modalité.

DONNEES BRUTES

Représentation des données de manière brute, sans unité particulière en ordonnées.

UNIFORMISATION

Uniformisation des données par un rapport entre la variable et le maximum de sa modalité.
